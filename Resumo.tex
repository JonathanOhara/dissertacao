\begin{resumo}
Esse trabalho explora a criação de uma API (Application Programming Interface) para um jogo de estratégia \textit{online} chamado Pokémon Showdown!, permitindo que pesquisadores criem agentes autônomos para competir contra jogadores humanos num ambiente extremamente competitivo com milhares de jogadores ativos a todo momento. A criação dessa API pode habilitar pesquisas de novas técnicas de inteligência artificial e comparação de diferentes técnicas quando submetidos a jogos com humanos.

Neste trabalho é discutido e desenvolvidos agentes para este jogo usando diferentes técnicas de árvores de busca de Monte Carlo. Importantes aspectos precisam ser levados em conta durante uma criação de agentes para esse jogo como: incompletude e imperfeição de informação e simultaneidade de escolhas são alguns dos principais desafios.
\end{resumo}
